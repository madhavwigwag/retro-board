# Retro Board 

## Create Board flow
    - Name Board. 
    - Add lists.
    - User is shown the board. 
    - User can share the link of the board. 

## Retro Board Experience
### Card Properties
    - Displays Name of the creator
    - Shows Comment 
    - Can be Up/Down Voted
    - Can be commented on
    - Can be deleted
    - Can be edited

### Profile
    - User can see boards he was a part of. 
    - user can create a board.
